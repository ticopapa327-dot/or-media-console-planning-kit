# 安全说明 / Security

## 中文

本仓库是公开文档仓库，不应包含任何敏感生产信息。

如果你发现以下内容，请不要在公开 issue 中贴出完整细节：

- 真实患者数据或可识别个人信息。
- 医院内部网络、账号、密钥、证书或接口凭据。
- 未脱敏的第三方厂商资料。
- 可能导致实际系统被攻击的漏洞利用细节。

请以最小必要信息描述问题，并先移除敏感内容。

## English

This is a public documentation repository and should not contain sensitive production information.

If you discover any of the following, please do not post full details in a public issue:

- Real patient data or personally identifiable information.
- Internal hospital network details, accounts, secrets, certificates, or interface credentials.
- Non-anonymized third-party vendor materials.
- Exploit details that could be used against a real system.

Please describe the issue with the minimum necessary information and remove sensitive details first.
