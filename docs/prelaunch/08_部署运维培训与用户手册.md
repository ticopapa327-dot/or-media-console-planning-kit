# 部署运维培训与用户手册

## 1. 部署前准备

| 项目 | 要求 |
| --- | --- |
| 网络 | 手术室媒体网、管理网、办公访问网分区；关键链路有线接入 |
| 服务器 | 核心服务、数据库、存储、监控和备份资源到位 |
| 设备 | 编码器、解码器、摄像头、显示器、音频矩阵、麦克风、扬声器清单确认 |
| 接口 | HIS/EMR/PACS/IAM/存储接口账号、证书、白名单和联调窗口确认 |
| 安全 | 防火墙、账号策略、证书、日志接入、漏洞扫描和等保要求确认 |
| 培训 | 用户名单、角色、房间授权、培训排期确认 |

## 2. 部署步骤

1. 安装服务器操作系统和基础依赖。
2. 配置网络、DNS、NTP、证书和防火墙策略。
3. 部署数据库、缓存、对象/文件存储和备份任务。
4. 部署核心服务、管理后台、客户端服务和媒体服务。
5. 配置房间、设备、输入源、输出源和默认场景。
6. 联调 HIS/EMR/PACS/IAM/存储/会议/设备接口。
7. 接入监控、日志、告警和审计归档。
8. 执行部署验证脚本和人工检查。
9. 完成管理员、用户和运维培训。
10. 进入试运行。

## 3. 回滚方案

- 应用回滚：保留上一版本镜像/安装包和配置备份。
- 数据库回滚：发布前备份，迁移脚本必须具备回滚或补偿方案。
- 设备配置回滚：导出发布前房间、设备、路由和场景配置。
- 业务回滚：必要时切回原有矩阵、录制或会议方案。
- 回滚触发：术中控制异常、数据丢失风险、安全事件、核心接口不可用。

## 4. 运维监控

| 监控项 | 告警条件 |
| --- | --- |
| 设备在线状态 | 编码器、解码器、摄像头、音频设备离线 |
| 媒体链路 | 丢包、延迟、码率异常、黑屏、无音频 |
| 存储 | 容量低于阈值、写入失败、备份失败 |
| 服务 | 服务不可用、重启频繁、CPU/内存/磁盘异常 |
| 安全 | 登录失败暴增、越权访问、异常导出、配置变更 |
| 审计 | 审计写入失败、日志归档失败 |

## 5. 维护计划

| 频率 | 任务 |
| --- | --- |
| 每日首次使用前 | 目测设备、线缆、接口、屏幕、音频设备，确认环境和电源 |
| 每次开机 | 自检核心服务、设备在线、存储容量、录制功能、会议功能 |
| 每日 | 检查告警、备份状态、存储容量；必要时重启非关键服务 |
| 每周 | 抽查审计日志、演练故障处理、检查接口队列 |
| 每月 | 安全补丁评估、容量趋势分析、恢复演练、账号复核 |
| 每季度 | 灾备演练、权限审计、培训复训、风险评审 |

## 6. 用户操作手册摘要

### 6.1 登录

1. 输入账号和密码，或使用医院统一身份。
2. 移动端和房间端选择授权房间。
3. 登录后确认房间、设备和存储状态正常。

### 6.2 视频路由

1. 进入视频路由页面。
2. 选择输入源。
3. 选择输出源。
4. 确认预览和连接状态。
5. 如需断开，点击已连接输出并确认。

### 6.3 PIP/PBP

1. 进入布局设置。
2. 选择 PIP 或 PBP。
3. 将输入源分配到布局窗口。
4. 保存或直接应用。

### 6.4 录制和截图

1. 从 HIS/EMR 查询患者，或手动新建患者/手术信息。
2. 选择录制输入源。
3. 点击开始录制。
4. 需要时暂停、恢复、静音、截图。
5. 点击停止后确认文件归档成功。

### 6.5 视频会议和示教

1. 创建会议并设置访问方式。
2. 邀请授权成员。
3. 管理麦克风、扬声器、成员静音和音量。
4. 结束后由会议创建者关闭会议。

### 6.6 文档管理

1. 按病历号、姓名、日期、术式、医生或手术间查询。
2. 展开患者/手术文件夹查看视频和图片。
3. 按权限回看、编辑、删除或下载。
4. 下载和删除必须确认并记录用途。

### 6.7 场景控制

1. 在场景设定中调整路由、摄像头、音频和布局。
2. 保存到我的场景并命名。
3. 使用前确认影响设备。
4. 点击应用场景。

## 7. 培训计划

| 对象 | 内容 | 考核 |
| --- | --- | --- |
| 手术室用户 | 登录、路由、录制、场景、异常提示 | 完成模拟手术流程 |
| 示教用户 | 房间观看、会议、文档回放 | 完成示教流程 |
| 管理员 | 账号、权限、设备、场景、审计 | 完成配置任务 |
| 运维人员 | 部署、监控、备份、恢复、故障处理 | 完成应急演练 |
| 信息科/安全 | 接口、安全策略、日志、合规 | 完成检查清单 |

## 8. 应急处理

| 事件 | 处理 |
| --- | --- |
| 黑屏或无信号 | 检查输入源、输出源、编码器、解码器、线缆和路由状态；必要时切换备用源 |
| 录制失败 | 检查存储容量、录制服务、输入源和患者关联；记录事件 |
| 会议异常 | 降级为只读直播或电话沟通；结束后导出日志 |
| 权限异常 | 立即禁用账号或会话，通知管理员和安全负责人 |
| 数据泄露疑似 | 启动安全事件响应，保全日志，暂停相关导出权限 |
