# 设备清单与电气连接设计

版本：0.1
日期：2026-06-29
输入资料：`数字化手术室连线图.vsdx`，并以 `数字化手术室连线图.pdf` 渲染图作为视觉核对。
公开说明：原始 VSDX/PDF 图纸作为本地输入资料使用，不作为公开仓库交付物。
适用范围：手术室端、示教报告厅、远程示教端、服务器侧。图纸中同时存在微创版、标准版、专业版示例，本文件按“标准版手术室端 + 示教/远程示教 + 1 台应用服务器 + 1 台存储服务器”整理。

## 1. 读取约定和限制

- VSDX 为主解析来源，保留了设备文字、形状和连接线；PDF 仅用于核对布局。
- 图纸线缆颜色含义：蓝色为视频线/HDMI 线，绿色为 LAN 网线，青色为音频线，黄色为光纤线，黑色为 USB 线。
- 图纸未明确展示强电供电、PDU、UPS、接地和机柜配电。本文件只整理弱电/信号/网络连接；强电方案需在实施设计中补充。
- 图纸未明确绘制“应用服务器”和“存储服务器”形状，本文件按项目约束补充服务器侧设备，并接入院内局域网/楼层交换机。
- 矩阵输入/输出口号以图纸为依据，现场实施时需与端口标签、跳线标签和系统配置三方核对。

## 7. 对软件开发的配置要求

| 配置对象 | 必填字段 |
| --- | --- |
| 房间 | 房间编码、房间类型、所属区域、上联交换机、启用状态 |
| 设备 | 设备编号、设备类型、IP/端口、线缆类型、所在房间、在线检测方式 |
| 视频源 | 源编号、源类型、编码板编号、矩阵输入口、预览流地址、是否可录制 |
| 视频输出 | 输出编号、显示设备、矩阵输出口、分辨率、是否支持远程示教 |
| 光纤链路 | 起点光端机、终点光端机、光纤编号、对应矩阵端口 |
| 音频设备 | 输入/输出类型、连接主机、功放/音箱关系、静音控制方式 |
| 服务器 | 应用服务器地址、存储服务器地址、存储协议、容量阈值、备份策略 |

## 8. 待现场确认项

- 8x8 矩阵各输入/输出端口的最终标签和设备对应关系。
- 36 进 4 出矩阵的输入口数量、输出口数量和每个光端机对应关系。
- 服务器部署位置、网段、VLAN、双网口/单网口、存储协议和容量规划。
